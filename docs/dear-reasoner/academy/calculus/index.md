## 词条1：函数——变化的数学模型

### 官方解释
**函数**是两个集合之间的对应关系：对于定义域中的每个输入，在值域中有唯一确定的输出。记作 $y = f(x)$，其中 $x$ 是自变量，$y$ 是因变量。

函数的表示方法：解析式、图像、表格、文字描述。

### 兔狲老师解释
函数就是"输入-输出机器"。

小小猪的比喻：像自动售货机，投币（输入）→ 出饮料（输出）。同样的硬币，总是出同样的饮料。

重要函数类型：
- 线性函数：$f(x) = ax + b$（直线）
- 二次函数：$f(x) = ax^2 + bx + c$（抛物线）
- 指数函数：$f(x) = a^x$（快速增长/衰减）
- 三角函数：$\sin x, \cos x$（周期性变化）

## 词条2：极限——无限接近的艺术

### 官方解释
**极限**描述函数在自变量趋近某个值时，函数值的变化趋势。记作：
$
\lim_{x \to a} f(x) = L
$
表示当 $x$ 无限接近 $a$ 时，$f(x)$ 无限接近 $L$。

极限的严格定义（ε-δ定义）：对于任意 $\epsilon > 0$，存在 $\delta > 0$，使得当 $0 < |x-a| < \delta$ 时，有 $|f(x)-L| < \epsilon$。

### 兔狲老师解释
极限就是'无限接近但永远不碰到'的数学描述。

小小猪的游戏：想象一个数不断接近2：1.9, 1.99, 1.999, 1.9999...
虽然永远到不了2，但可以**无限接近**。

重要极限：
$
\lim_{x \to 0} \frac{\sin x}{x} = 1
$
$
\lim_{x \to \infty} \left(1 + \frac{1}{x}\right)^x = e
$

## 词条3：连续性——没有断裂的变化

### 官方解释
函数 $f(x)$ 在点 $x=a$ 处**连续**，如果满足：
1. $f(a)$ 有定义
2. $\lim_{x \to a} f(x)$ 存在
3. $\lim_{x \to a} f(x) = f(a)$

直观上：函数图像在 $x=a$ 处没有断裂，可以一笔画过。

### 兔狲老师解释
连续就是'不断开的变化'。

小小猪的比喻：连续函数像平滑的道路，可以顺畅行驶；不连续函数像有断崖的道路，需要跳跃。

连续性的重要性：微积分基本定理要求函数连续。

常见不连续类型：
- 可去间断点：极限存在但不等于函数值
- 跳跃间断点：左右极限存在但不相等
- 无穷间断点：极限为无穷大

## 词条4：导数——瞬时变化率

### 官方解释
函数 $f(x)$ 在点 $x=a$ 处的**导数**定义为：
$
f'(a) = \lim_{h \to 0} \frac{f(a+h) - f(a)}{h}
$
如果这个极限存在。

导数的几何意义：曲线 $y=f(x)$ 在点 $(a, f(a))$ 处切线的斜率。

### 兔狲老师解释
导数就是'瞬间的速度表'。

小小猪的自行车实验：
- 平均速度：总路程 ÷ 总时间
- 瞬时速度：**某一瞬间**的速度表读数

导数就是数学版的'瞬间速度表'！

导数的物理意义：
- 位置函数的导数 = 速度
- 速度函数的导数 = 加速度
- 经济：成本函数的导数 = 边际成本

## 词条5：导数的计算规则

### 官方解释
基本求导公式：
1. 常数：$(c)' = 0$
2. 幂函数：$(x^n)' = nx^{n-1}$
3. 指数函数：$(e^x)' = e^x$, $(a^x)' = a^x \ln a$
4. 对数函数：$(\ln x)' = \frac{1}{x}$, $(\log_a x)' = \frac{1}{x \ln a}$
5. 三角函数：$(\sin x)' = \cos x$, $(\cos x)' = -\sin x$

求导法则：
1. 和差法则：$(f \pm g)' = f' \pm g'$
2. 积法则：$(fg)' = f'g + fg'$
3. 商法则：$\left(\frac{f}{g}\right)' = \frac{f'g - fg'}{g^2}$

### 兔狲老师解释
这些规则是'导数计算的工具箱'。

小小猪的学习方法：
- 幂法则：$x^3$ 的导数是 $3x^2$（指数降1，乘原指数）
- 积法则：像分配律但有交叉项
- 商法则：分子像积法则但减号，分母平方

记忆技巧：$\sin$ 导得 $\cos$，$\cos$ 导得 $-\sin$，正负交替。

## 词条6：链式法则——复合函数的导数

### 官方解释
**链式法则**：如果 $y = f(g(x))$，那么
$
\frac{dy}{dx} = f'(g(x)) \cdot g'(x)
$
或者记作 $\frac{dy}{dx} = \frac{dy}{du} \cdot \frac{du}{dx}$，其中 $u = g(x)$。

### 兔狲老师解释
链式法则就是'洋葱剥皮法'！

小小猪的比喻：$y = \sin(x^2)$ 像洋葱：
- 外层：$\sin(\text{东西})$
- 内层：$x^2$
- 剥皮：先处理外层导数，再乘内层导数

或者像工厂流水线：原材料 $x$ → 加工1（$x^2$）→ 加工2（$\sin$）→ 成品 $y$。
每个加工步骤都会改变材料。

## 词条7：高阶导数——变化的变化

### 官方解释
**高阶导数**是导数的导数：
- 一阶导数：$f'(x)$ 或 $\frac{dy}{dx}$
- 二阶导数：$f''(x)$ 或 $\frac{d^2y}{dx^2}$
- $n$阶导数：$f^{(n)}(x)$ 或 $\frac{d^ny}{dx^n}$

二阶导数的物理意义：加速度是速度的导数，是位置的二阶导数。

### 兔狲老师解释
高阶导数就是'变化的深度分析'。

小小猪的运动分析：
- 位置 $s(t)$：你在哪里
- 速度 $v(t) = s'(t)$：你移动多快
- 加速度 $a(t) = v'(t) = s''(t)$：你的速度变化多快
- 急动度 $j(t) = a'(t) = s'''(t)$：你的加速度变化多快（坐过山车的感觉！）

几何意义：
- 一阶导数：切线斜率
- 二阶导数：曲线的凹凸性
  - $f''(x) > 0$：凹向上（像碗）
  - $f''(x) < 0$：凹向下（像拱）

## 词条8：隐函数求导与相关变化率

### 官方解释
**隐函数求导**：当函数关系由方程 $F(x,y)=0$ 隐含给出时，对等式两边关于 $x$ 求导，然后解出 $\frac{dy}{dx}$。

**相关变化率**：两个相关变量 $x$ 和 $y$ 都随时间变化，已知一个的变化率，求另一个的变化率。

### 兔狲老师解释
隐函数求导就是'方程中的求导游戏'。

小小猪的例子：圆方程 $x^2 + y^2 = 25$
- 显式：$y = \sqrt{25 - x^2}$（上半圆）
- 隐式：$x^2 + y^2 = 25$（整个圆）

对隐式求导：$2x + 2y \frac{dy}{dx} = 0$ ⇒ $\frac{dy}{dx} = -\frac{x}{y}$

相关变化率就像'连动的齿轮'：一个齿轮转得快慢影响另一个齿轮。

## 词条9：微分——变化的线性近似

### 官方解释
函数 $y=f(x)$ 在 $x=a$ 处的**微分**定义为：
$
dy = f'(a) dx
$
其中 $dx$ 是自变量的微小变化，$dy$ 是因变量的近似变化。

几何意义：用切线近似曲线。

### 兔狲老师解释
微分就是'放大镜下的近似'。

小小猪的比喻：在非常小的范围内，曲线看起来像直线。
- 精确变化：$\Delta y = f(a+dx) - f(a)$
- 近似变化：$dy = f'(a) dx$

当 $dx$ 很小时，$dy \approx \Delta y$。

应用：工程计算、误差估计、数值方法。

## 词条10：积分——累积的艺术

### 官方解释
**定积分**：函数 $f(x)$ 在区间 $[a,b]$ 上的定积分定义为黎曼和的极限：
$
\int_a^b f(x) dx = \lim_{n \to \infty} \sum_{i=1}^n f(x_i^*) \Delta x
$
几何意义：曲线 $y=f(x)$ 下方，$x=a$ 到 $x=b$ 之间的面积。

**不定积分**（原函数）：$F'(x) = f(x)$，则 $F(x)$ 是 $f(x)$ 的不定积分，记作 $\int f(x) dx = F(x) + C$。

### 兔狲老师解释
如果导数是'瞬间速度表'，积分就是'里程表'！

小海豹的历史视角：阿基米德用穷竭法求面积，是积分的雏形。

积分的物理意义：
- 速度函数的积分 = 总路程
- 力函数的积分 = 总功
- 密度函数的积分 = 总质量

不定积分和定积分的关系：微积分基本定理！

## 词条11：微积分基本定理——导数和积分的桥梁

### 官方解释
**微积分基本定理**包含两部分：

第一部分：如果 $f$ 在 $[a,b]$ 上连续，$F(x) = \int_a^x f(t) dt$，则 $F'(x) = f(x)$。

第二部分：如果 $F$ 是 $f$ 的一个原函数，则
$
\int_a^b f(x) dx = F(b) - F(a)
$

### 兔狲老师解释
这是'微积分的核心奇迹'！

小小猪的发现：导数和积分是**互逆运算**，就像加法和减法、乘法和除法。

比喻：如果导数是从电影中提取一帧（瞬间），积分就是把所有帧连成电影（全过程）。

微积分基本定理的意义：
1. 连接了微分和积分
2. 提供了计算定积分的实际方法
3. 解释了为什么求面积和求原函数相关

## 词条12：微分方程——描述变化的方程

### 官方解释
**微分方程**是包含未知函数及其导数的方程。

分类：
- 常微分方程（ODE）：未知函数是一元函数
- 偏微分方程（PDE）：未知函数是多元函数

阶数：方程中最高阶导数的阶数。

例子：
- 一阶线性ODE：$\frac{dy}{dx} + P(x)y = Q(x)$
- 二阶常系数齐次ODE：$ay'' + by' + cy = 0$

### 兔狲老师解释
微分方程就是'变化的规则说明书'！

小小猪的发现：知道变化率（导数），想求函数本身，就需要解微分方程。

重要应用：
1. **指数增长/衰减**：$\frac{dP}{dt} = kP$ ⇒ $P(t) = P_0 e^{kt}$
   - 人口增长、放射性衰变、连续复利
   
2. **弹簧振动**：$m\frac{d^2x}{dt^2} + c\frac{dx}{dt} + kx = 0$
   - 无阻尼：简谐振动 $x(t) = A\cos(\omega t + \phi)$
   - 有阻尼：振幅衰减的振动

3. **逻辑斯蒂方程**：$\frac{dP}{dt} = rP(1 - \frac{P}{K})$
   - 考虑环境承载力的种群增长

1. **函数** → 变化的数学模型
2. **极限** → 无限接近的严格定义  
3. **连续性** → 没有断裂的变化
4. **导数** → 瞬时变化率
5. **求导规则** → 导数的计算方法
6. **链式法则** → 复合函数的导数
7. **高阶导数** → 变化的变化
8. **隐函数求导** → 方程中的导数
9. **微分** → 线性近似
10. **积分** → 累积求和
11. **基本定理** → 导数与积分的桥梁
12. **微分方程** → 描述变化的方程

**兔狲教授的最后一课**：
"微积分不是一堆孤立的技巧，而是一个有机的整体。从极限定义导数，用导数研究变化，用积分累积变化，最后用微分方程描述变化规律——这是一个完整的思维体系。

更重要的是，微积分教会我们一种思考方式：在变化中寻找规律，在瞬间中看到永恒，在局部中把握整体。这才是推理科学家的核心能力。

现在，你不仅学会了微积分的技术，更掌握了分析变化的思维工具。用这些工具去理解世界吧——从物理运动到经济增长，从生物进化到人工智能，变化无处不在，而微积分是理解变化的语言。"
